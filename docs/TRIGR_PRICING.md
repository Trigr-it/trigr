# TRIGR — Pricing & Business Model
> Current confirmed pricing as of March 2026.

---

## Tier Structure

### Free — Free forever
Everything a personal user needs. No artificial limits on core functionality.

- Unlimited hotkeys on any key combination
- Unlimited text expansions with fill-in variables
- Unlimited app-specific profiles with automatic switching
- AHK Script Runner (v1 + v2, bundled runtime, no AHK install needed)
- List view toggle (keyboard on by default, togglable)
- Quick Search (Ctrl+Space overlay)
- Save Snippet from Highlight — NO, moved to Pro
- Clipboard Manager basic — history, search, pin, 30-day retention
- Basic analytics — total actions fired + time saved counter
- Starter templates
- Profile export/import (file-based)
- Macro recorder
- Auto-updater

### Pro — £49/year or £99 lifetime
Advanced automation features for power users.

- Everything in Free, plus:
- Clipboard Manager advanced (smart type detection, source app tagging, sensitive item detection, unlimited retention)
- Save Snippet from Highlight (Ctrl+Space contextual card)
- Full analytics dashboard (14-day chart, per-assignment breakdown, CSV export)
- AHK importer (.ahk file parser)
- TextExpander importer
- espanso importer
- Conditional expansions
- Scheduled macros / time triggers
- Mouse button assignments (advanced)
- Dynamic variables ({date}, {time}, {computername}, {username})
- Regex triggers
- Priority support (24-hour response)

### Teams — £12-15/user/month
Primary revenue engine. Cloud sync and shared libraries for businesses.

- Everything in Pro, plus:
- Cloud sync (assignments and profiles across all devices)
- Shared snippet libraries (team-wide, managed centrally)
- Admin dashboard (users, licences, shared libraries)
- Team analytics (usage across all team members)
- Centralised profile management (push to all team members)
- SSO (Microsoft, Google)
- Priority onboarding (dedicated setup call)

## Beta Strategy

LemonSqueezy integrated during Beta with free £0 keys.  
- Testers receive a free licence key via LemonSqueezy
- Validates the full licence flow before any money changes hands
- At v1.0, paid products added alongside — infrastructure already live
- Natural upgrade: "your beta key is expiring — here's 20% off Pro to continue"
- Keys can be individually revoked via LemonSqueezy dashboard

---

## Revenue Logic

- Free tier removes adoption barrier — no one refuses a free tool
- Free individual users become advocates who bring Trigr into their workplaces
- 10 Teams clients at 20 users at £12/month = £28,800/year
- 1 enterprise client at 50 users = £7,200/year alone
- Teams revenue scales per user — Pro revenue is fixed per subscriber
- Lifetime at £99 generates upfront cash to fund Teams infrastructure
