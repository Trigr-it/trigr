## CC Session Definition

1 CC session = one continuous Claude Code conversation  
Rory's time per session: 20-40 minutes  
(Claude Chat writes prompts, Rory copy/pastes into CC, pastes result back — Rory writes no code)  
Calendar time: roughly 1-2 days per 4-6 sessions
