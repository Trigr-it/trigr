Trigr User Guide

Trigr is a visual hotkey, macro and text expansion manager that runs entirely on your local machine. All your data stays on your device — nothing is ever transmitted to any server.

Overview

Trigr has three main capabilities:

  • Hotkey Mapping — assign macros to any key combination (Ctrl+Alt+K, Win+F2, etc.) or mouse button
  • Text Expansions — type a short trigger word and have it automatically replaced with a longer phrase, template or dynamic text
  • Autocorrect — automatically fix common typos as you type, system-wide

Everything fires globally in the background — macros and expansions work in any application.

Interface

Title Bar

The title bar contains the profile switcher, a Macros On/Off toggle, theme toggle, and the Settings button. You can pause all macros instantly with the toggle without closing the app.

Sidebar

The sidebar switches between Mapping (keyboard/mouse assignment view) and Text Expansions.

Modifier Bar

The row of buttons above the keyboard — Ctrl Alt Shift Win — lets you select which hotkey layer you're viewing. Click one or more to see what's assigned to that combination. Only keys assigned on that layer are highlighted.

Hotkey Mapping

  1. Click one or more modifier buttons in the modifier bar (e.g. Ctrl)
  2. Click any key on the keyboard that you want to assign
  3. In the panel on the right, choose what the hotkey should do: launch an app, open a URL, type text, run a macro sequence, etc.
  4. Click Save
Tip: Use the ⏺ Record button to capture a hotkey by simply pressing it on your keyboard, instead of clicking modifier buttons manually.

Double-tap support

Any hotkey can have a second action assigned to it that fires when you press the same combo twice quickly. In the macro panel, switch to the ×2 Double-tap tab to assign a separate action.

Bare Keys

In app-specific profiles, you can assign macros to bare keys (no modifier required) that only fire when that app is focused. Select Bare Keys in the modifier bar — this option is only available in profiles with a linked application.

Profiles

Profiles let you have different hotkey sets for different contexts. The profile switcher is in the title bar.

Global profile

The active global profile fires its macros in all applications, unless an app-specific profile takes precedence.

App-specific profiles

Create a profile and link it to a specific application (e.g. AutoCAD, Photoshop). When that application becomes the foreground window, Trigr automatically switches to that profile. When you switch away, it switches back.

  1. Click the profile name in the title bar to open the profile menu
  2. Click + New Profile
  3. In Settings for that profile, click Link to Application and browse for the executable

Text Expansions

Text expansions let you type a short word (the trigger) and have it automatically replaced with longer text. They work in any application, system-wide.

  1. Click Text Expansions in the sidebar
  2. Click + Add
  3. Enter a trigger word (e.g. signoff) — no spaces allowed
  4. Optionally add a display name so you can find it easily in search
  5. Type your replacement text in the editor
  6. Click Save

Type your trigger word followed by Space in any app and it will be replaced instantly. For ⚡ Instant mode, the expansion fires immediately after the last character without needing Space.

Tip: Use categories to organise expansions — add a category from the bar above the list, then assign expansions to it.

Dynamic tokens

Click Insert ▾ in the editor toolbar to add dynamic content:

TokenWhat it inserts
{clipboard}The current clipboard contents at time of firing
{date:DD/MM/YYYY}Today's date in the chosen format
{time:HH:MM}The current time
{dayofweek}Day name e.g. "Monday"
{cursor}Moves the cursor to this position after pasting
Fill-in Field…Prompts you to type a value each time the expansion fires

Autocorrect

Trigr includes a built-in library of 50+ common typos (teh→the, recieve→receive, etc.) plus a custom corrections list you manage yourself.

Open Text Expansions → Autocorrect tab to enable the built-in library and add your own custom corrections.

Quick Search

Quick Search is a floating overlay that lets you find and fire any macro or expansion from any application without switching to Trigr.

  • Default hotkey: Ctrl+Space
  • Type to search across all your macros and expansions
  • Press Enter or click to fire the selected result
  • Press Escape to dismiss

You can change the hotkey in Settings → Quick Search.

Note: Win+Space is reserved by Windows for input language switching and cannot be used as the Quick Search hotkey.

Tokens & Fill-in Fields

Fill-in fields let you create template expansions where you're prompted to type a value each time the expansion fires. For example, a "thank you email" expansion might have a fill-in for the recipient's name.

  1. In the expansion editor, click Insert ▾ → Fill-in Field…
  2. Type a label for the prompt (e.g. "Recipient Name")
  3. Click Insert

When the expansion fires, a small popup will appear asking you to fill in each field. Press Tab to move between fields and Enter to submit.

Recording Hotkeys

Instead of clicking modifier buttons manually, you can press ⏺ Record in the modifier bar and then press the actual key combination you want to assign. Trigr will capture the modifiers and key and select that layer automatically.

  • Press Escape during recording to cancel
  • The keyboard dims while recording is active

Mouse Button Macros

Switch to the Mouse view in the sidebar to assign macros to mouse buttons (back/forward buttons, middle click, etc.). The same modifier layers apply — hold Ctrl and click a mouse button to assign a Ctrl+MouseButton macro.

Backup & Restore

Trigr automatically backs up your config on every launch and every save. Backups are stored in:

%APPDATA%\KeyForge\backups\

The last 10 timestamped backups are kept, plus a keyforge-config-last-known-good.json that is always the most recent valid save.

Restoring from backup

  1. Open Settings → Backup & Restore
  2. Click Restore from Automatic Backup…
  3. Choose a backup from the list — each shows date, profile count and expansion count
  4. Click Restore and confirm

You can also manually export your config from Settings and import it on another machine.

Troubleshooting

ProblemSolution
Hotkeys not firing Check the Macros toggle in the title bar — it may be paused. Also ensure the hotkey combo doesn't conflict with a system shortcut.
Text expansions not expanding Make sure you typed the trigger word exactly and followed it with Space. Check that Autocorrect or the target app isn't intercepting the trigger first.
Quick Search hotkey not responding Win+Space is reserved by Windows. Try Ctrl+Space or Ctrl+Shift+Space. Change it in Settings → Quick Search.
Wrong text pasted in CAD/games Change the input method in Settings → Compatibility → Global input method. Try Direct keystrokes for CAD software.
Fill-in popup doesn't appear The fill-in window may be behind other windows. Check your taskbar. It appears near the cursor position.
Config seems empty after update Open Settings → Backup & Restore and restore from a recent automatic backup.
